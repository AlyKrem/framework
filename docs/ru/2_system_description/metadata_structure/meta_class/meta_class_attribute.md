# Мета классов - атрибутивная часть
### Предыдущая страница: [Мета классов - общая часть](/docs/ru/2_system_description/metadata_structure/meta_class/meta_class_main.md)

**Атрибутивная часть меты класса** описывает поля атрибута класса. Атрибутов в классе обычно минимум два - ключевое поле и поле данных. Они содержатся в виде массива в поле `"properties"` основной части меты классов. Каждый атрибут - это объект следующей структуры:

## JSON
```
    {
      "orderNumber": 20,
      "name": "integer_integer",
      "caption": "Редактор целых чисел [14]",
      "type": 6,
      "size": null,
      "decimals": 0,
      "allowedFileTypes": null,
      "maxFileCount": 0,
      "nullable": true,
      "readonly": false,
      "indexed": false,
      "unique": false,
      "autoassigned": false,
      "hint": null,
      "defaultValue": null,
      "refClass": "",
      "itemsClass": "",
      "backRef": "",
      "backColl": "",
      "binding": "",
      "semantic": null,
      "selConditions": [],
      "selSorting": [],
      "selectionProvider": null,
      "indexSearch": false,
      "eagerLoading": false,
      "formula": null,
      "cached": true
    }
```
## Описание полей

| Код                   | Имя                   | Допустимые значения                  | Описание                                                                                                                                                                                                                                                                 |
|:-----------------------|:-------------------------------------------|:-------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `"orderNumber"`       | **Порядковый номер**                       | Целое неотрицательное                | Задает расположение атрибута относительно других атрибутов этого же класса.                                                                                                                                                                |
| `"name"`               | **Системное имя**                          | Строка, только латиница без пробелов | Указывается имя атрибута, с которым будет работать система, а значит не может быть пустым, может содержитать только символовы латинского алфавита, без пробелов (задается один раз при создании атрибута).                                                     |
| `"caption"`            | **Логическое имя**                         | Строка                               | Отображение имени атрибута в пользовательском интерфейсе.                                                                                                                                                                                                                |
| [`"type"`](/docs/ru/2_system_description/metadata_structure/meta_class/property_types.md)               | **Тип**                                    | Целое - идентификатор (код) типа     | Тип данных атрибута.                                                                                                                                                                                                                   |
| `"size"`               | **Размер**                                 | Целое положительное                  | Максимальный размер данных атрибута, допистимые значения зависят от типа атрибута.                                                                                                                                                                                       |
| `"decimals"`           | **Число знаков после запятой**             | Целое неотрицательное                | Число знаков после запятой, задается только для типа "Десятичное [8]".                                                                                                                                                                                                   |
| `"allowedFileTypes"`   | **Допустимые типы файлов**                 | Массив строк                         | Позволяет задать допустимые расширения файлов, которые пользователь может загрузить в атрибут типа "Коллекция файлов [110]".                                                                                                                                             |
| `"maxFileCount"`       | **Максимальное количество файлов**         | Число от 1 до 5 (максимум 5 файлов)  | Задает максимальное количество файлов, которые пользователь может загрузить в атрибут типа "Коллекция файлов [110]".                                                                                                                                                     |
| `"nullable"`           | **Допустимо пустое значение**              | Логический                           | Разрешает или запрещает пустое значение атрибута.                                                                                                                                                                                                                        |
| `"readonly"`           | **Только для чтения**                      | Логический                           | Разрешает или запрещает изменять значение атрибута.                                                                                                                                                                                                                      |
| `"indexed"`            | **Индексировать для поиска**               | Логический                           | Указывает, нужно ли индексировать значения данного атрибута для ускорения поиска. [Подробнее] (metadata/indexed)                                                                                                                                                                                       |
| `"unique"`             | **Уникальные значения**                    | Логический                           | Для атрибута накладывает ограничение уникальности (нельзя создать два объекта класса с одинаковыми значениями в уникальном атрибуте).                                                                                                                                    |
| [`"autoassigned"`](/docs/ru/2_system_description/metadata_structure/meta_class/atr_autoassigned.md).       | **Автозаполнение**                         | Логический                           | Разрешает или запрещает автоматическое заполнение поля приложением.
| `"hint"`               | **Подсказка**                              | Строка                               | Задает сообщение, которые выведется в пользовательском интерфейсе рядом с именем атрибута.                                                                                                                                                                               |
| `"defaultValue"`      | **Значение по умолчанию**                  | Зависит от типа атрибута             | Указывается значение, которое будет заполнено в атрибуте в форме создания (при создании объекта). [Подробнее] (metadata/defaultValue)                                                                                                                                                                       |
| [`"refClass"`](/docs/ru/2_system_description/metadata_structure/meta_class/atr_ref_backref.md)          | **Класс ссылки**                           | Строка, только латиница, без пробелов | Содержит значение поля `"name"` (Системное имя) класса, который должен использоваться в атрибуте типа "Ссылка [13]". [Подробнее](metadata/reference).                                                                                                                    |
| [`"itemsClass"`](/docs/ru/2_system_description/metadata_structure/meta_class/atr_itemclass_backcoll.md)          | **Класс коллекции**                        | Строка, только латиница без пробелов | Содержит значение поля `"name"` (Системное имя) класса, объекты которого могут привязаться к атрибуту типа "Коллекция [14]"                                                                                                         |
| [`"backRef"`](/docs/ru/2_system_description/metadata_structure/meta_class/atr_ref_backref.md)          | **Атрибут обратной ссылки**                | Строка, только латиница без пробелов | Указывается атрибут типа "Ссылка [13]", из класса, указанного в свойстве Класс коллекции, который ссылается на исходный класс. Нужно для фильтрации и привязки объектов из класса Класс коллекции по значению ссылочного атрибута.      |
| [`"backColl"`](/docs/ru/2_system_description/metadata_structure/meta_class/atr_itemclass_backcoll.md)          | **Атрибут обратной коллекции**             | Строка, только латиница без пробелов | Указывается атрибут типа "Коллекция [14]", из класса, указанного в свойстве Класс коллекции, который ссылается на исходный класс. Нужно для фильтрации и привязки объектов из класса Класс коллекции по значению ссылочного атрибута.  [Подробнее] (metadata/collection) |
| `"binding"`            | **Основание коллекции**                    | Строка, только латиница без пробелов | В реестре разработка не была завершена (регистри - нет данных). Указывается атрибут класса, к которому привязывается атрибут обратной ссылки. Если не указан, то принимается ключевой атрибут.                                                                           |
| [`"semantic"`](/docs/ru/2_system_description/metadata_structure/meta_class/atr_semantic.md)           | **Семантика**                              | Строка                               | Указывается для ссылочных атрибутов для того, чтобы выводить и формировать информацию из ссылочного класса, корректную для описания артибута в объекте исходного класса.                                                               |
| [`"selConditions"`](/docs/ru/2_system_description/metadata_structure/meta_class/atr_selconditions.md)     | **Условия отбора допустимых значений**     | Null либо массив объектов            | Позволяет ограничить выбор объектов по ссылке, допустимых для привязкки в данном ссылочном атрибуте.                                                          |
| [`"selSorting"`](/docs/ru/2_system_description/metadata_structure/meta_class/atr_selsorting.md)        | **Сортировка выборки допустимых значений** | Null либо массив объектов            | Позволяет сортировать выбор объектов по ссылке, допустимых для привязкки в данном ссылочном атрибуте.                                                                  |
| [`"selectionProvider"`](/docs/ru/2_system_description/metadata_structure/meta_class/atr_selectionprovider.md) | **Список выбора допустимых значений**      | Null либо объект                     | Задает список выбора допустимых значений для атрибута.                                                                                                                                                                         |
| [`"indexSearch"`](/docs/ru/2_system_description/metadata_structure/meta_class/atr_indexed.md)       | **Полнотекстовый поиск**                   | Логический                           | Признак использования атрибута в полнотекстовом поиске. Указывает, что значение данного атрибута должно индексироваться поисковой системой.                                                                                                                              |
| [`"eagerLoading"`](/docs/ru/2_system_description/metadata_structure/meta_class/eager_loading.md)      | **Жадная загрузка**                        | Логический                           | Загрузка объекта по ссылке (для атрибутов типа ссылка и коллекция).                                                                                                                                                                                                      |
| [`"formula"`](/docs/ru/2_system_description/metadata_structure/meta_class/atr_formula.md)            | **Вычислимые поля**                        | Null либо объект                     | Указывает на формулу расчета.                                                                                                                                                                                                             |
| [`"cached"`](/docs/ru/2_system_description/metadata_structure/meta_class/atr_cached_true.md)            | **Кеширования значения вычислимого атрибута**                        | Логический                     | Применяется только для значений атрибута, полученных с помощью вычисления по формуле. Указывает на возможность кеширования значения вычислимого атрибута.                                                                                                                                                                                                             |
