# Мета классов - общая часть
**Общая часть меты классов** - содержит поля параметров класса, которые имеют отношения к самой структуре и способам оперирования данными в ней.
## JSON
```
{
  "isStruct": false,
  "key": [
    "id"
  ],
  "semantic": "",
  "name": "class_integer",
  "abstract": true,
  "version": "",
  "caption": "Класс \"Целое [6]\"",
  "ancestor": null,
  "container": null,
  "creationTracker": "",
  "changeTracker": "",
  "creatorTracker": "",
  "editorTracker": "",
  "history": 0,
  "journaling": false,  
  "compositeIndexes": null,
  "properties": [...]
}
```
## Описание полей

|&ensp; Код      &ensp;           | Имя          | Допустимые значения                    |  Описание                                                                                                                                                                                                 |
|:--------------------:|:-------------------------------:|:--------------------------------------:|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [`"isStruct"`](/metadata_structure/meta_class/type_isstruct16.md)      &ensp;  | **Является структурой**         | Логическое.                            | Если выставлено "true" - это говорит о том, что данный класс является структурой, и может быть использован в прочих классах в атрибутах особого типа - Структура [16] |
| [`"key"`](/metadata_structure/meta_class/key.md)           &ensp;   | **Ключевые атрибуты**           | Массив строк, минимум одно значение.   | Для функционирования приложения в каждом классе должно быть задано ключевое поле, однозначно идентифицирующее объект в коллекции.                                            |
| [`"semantic"`](metadata_structure/meta_class/semantic.md)    &ensp;     | **Семантический атрибут**       | Строка.                                | Задает семантику - правило формирования строкового представления для данного класса.                                                                                    |
| `"name"`     &ensp;        | **Системное имя**               | Строка, только латиница, без пробелов. | Задает в том числе первую часть имени файла меты класса, служебное имя.                                                                                                                                  |
| [`"abstract"`](/metadata_structure/meta_class/abstract.md)     &ensp;        | **Признак абстрактности для класса**               | Логическое | Используется только для родительских (базовых) классов.                                                                                                                                  |
| [`"version"`](/metadata_structure/meta_class/metaverion.md)    &ensp;      | **Версия**                      | Строка.                       | Позволяет задавать версионирвоание меты, для возмжности оперирования данными созданными в разных версиях меты в рамках одной коллекции.                                    |
| `"caption"`     &ensp;     | **Логическое имя**              | Строка.                                | Отображаемое в пользовательском интерфейсе имя класса                                                                                                                                                    |
| [`"ancestor"`](/metadata_structure/meta_class/ancestor.md)    &ensp;     | **Родительский класс**          | Null либо строка.                      | Набор атрибутов, заведенных в данном классе, наследуется классами-наследниками. Является способом сократить количество сущностей, когда для них можно использовать одинаковый набор атрибутов. Все классы-наследники будут наследовать атрибутивный состав родителя + можно завести атрибуты, принадлежащие индивидуально данному классу-наследнику (при необходимости)                                                                                                                                                                                        |
| `"container"`    &ensp;    | **Атрибут ссылки на контейнер** | Null либо строка.                      | **НЕТ ДАННЫХ**                                                                                                                                                                                           |
| [`"creationTracker"`](/metadata_structure/meta_class/time_user_tracker.md) &ensp; | **Метка времени создания**      | Строка                                 | Позволяет сохранять в классе дату/время создания объекта, требует наличия соответствующего атрибута класса, `"name"` которого и вносится в данное поле.             |
| [`"changeTracker"`](/metadata_structure/meta_class/time_user_tracker.md) &ensp;   | **Метка времени изменения**     | Строка                                 | Позволяет сохранять в классе дату/время изменения объекта, требует наличия соответствующего атрибута класса, `"name"` которого и вносится в данное поле.              |
| [`"creatorTracker"`](/metadata_structure/meta_class/time_user_tracker.md) &ensp;  | **Метка пользователя создавшего объект**      | Строка                                 | Позволяет сохранять в классе имя пользователя создавшего объект, требует наличия соответствующего атрибута класса, `"name"` которого и вносится в данное поле.               |
| [`"editorTracker"`](/metadata_structure/meta_class/time_user_tracker.md) &ensp;   | **Метка пользователя изменившего объект**     | Строка                                 | Позволяет сохранять в классе имя пользователя, изменившего объект, требует наличия соответствующего атрибута класса, `"name"` которого и вносится в данное поле.            |
| `"history"`     &ensp;     | **Снимки данных**               | _0 - нет_                              | Хранит режим снимков данных                                                                                                                                                                              |
|                      |                                 | _1 - произвольно_                      |                                                                                                                                                                                                          |
|                      |                                 | _2 - с точностью до часа_              |                                                                                                                                                                                                          |
|                      |                                 | _3 - с точностью до суток_             |                                                                                                                                                                                                          |
|                      |                                 | _4 - с точностью до недели_            |                                                                                                                                                                                                          |
|                      |                                 | _5 - с точностью до месяца_            |                                                                                                                                                                                                          |
|                      |                                 | _6 - с точностью до года_              |                                                                                                                                                                                                          |
| [`"journaling"`](/metadata_structure/meta_class/journaling.md)  &ensp;     | **Журналирование изменений**    | Логическое.                            | Разрешает/запрещает журналирование изменений объектов класса.                                                                                                            |
| [`"compositeIndexes"`](/metadata_structure/meta_class/compositeindexes.md) &ensp; | **Индексация**                  | Null либо массив объектов.             | Позволяет задать требования уникальности сочетания полей.                                                                                                          |
| [`"properties"`](/metadata_structure/meta_class/meta_class_attribute.md)   &ensp;    | **Атрибуты**                    | Массив объектов.                       | Массив атрибутов сущности. Каждый объект описывается в соответствии с атрибутивной частью меты классов.
